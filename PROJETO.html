O telégrafo foi a primeira forma de comunicação a distancia depois da carta. Em 1958 foi colocado um cabo intercontinental com o código morse no fundo do oceano.

1950 Guerra Fria - ENIAC veio do DARPA (Calculava rota de mísseis)

1960 Joseph Carl Robert Lickder teorizou uma rede que permitiria a todos se comunicarem.

Arpanet - Sistema de comunicação em pacotes, enviados um a um e que permita múltiplos destinos (Estudos sobre nós) Pontos de intersecção de informações funcionando como "Checkpoints". Podemos dizer que é o "avô dos roteadores".

Vint Cerf e Robert Kahn criaram o protocolo TCP/IP Ganhou mais robustez e criando micro-redes com o tempo com MILNET, NSFNet, NASA, ESNet, CSNet.

Em 1989, Tim Berners-Lee viu a oportunidade de unir hipertexto com TCP/IPe criou a WWW (World Wide Web), o primeiro navegador. Popularizou o HTML e HTTP e foi diretor do W3c.

Marc Andresseen tornou o navegador mais amigável com recursos gráficos. Reinventou o navegador Netscapenativator feito a partir do Mosaic. Deteve 90% da internet na época, mas sendo superado pelo Internet Explorer anos mais tarde.

Bill Gates fundou a Microsoft e criou o Internet Explorer.

Conceito de Client no linguajar Web

Dispositivo que está consumindo informação na Internet.

O servidor retorna arquivos estáticos em forma de JS, HTML, CSS, imagens (depende da velocidade de conexão e do tamanho dos arquivos). Após isso, o navegador armazena os arquivos recebidos em forma de cache. O navegador salva para não precisar baixar tudo de novo quando você atualiza a página.

Navegadores

São programas usados por empresas que servem para abrir/executar arquivos; Seguem padrões W3C e também é chamado de Browser. Iniciou com o Mosaic, passou para o Netscape e hoje existe uma infinidade deles.

Aplicação Web

São poluções criadas que usam a internet como meio de comunicação entre Client X Server não sendo necessário a sua instalação.

Quando usamos no navegador são aplicações web. Quando está instalado no dispositivo é no sistema operacional.

Sobre Servidores

(Tipos de servidores)

  1. Servidor de Arquivos;
  2. Servidor de segurança (Firewall, por exemplo);
  3. Servidor para Streaming (Geralmente de vídeos);
  4. Servidor de E-mails (armazena e gerencia envios de emails)
  5. Servidor web (html, css, js)

DATACENTERS

São servidores enormes que servem para armazenar muita coisa, geralmente de empresas grandes como o Google, Meta, etc.

Servidor Proxy

É um servidor dentro de outro servidor que possui algumas regras de restrição de acesso. Se consegui acessar por fora da rede é porque o proxu está bloqueando. Igual o proxy que bloqueia o youtube no trabalho.

Firewall

Serve para acesso externo. Barreira de fogo de segurança. É um sistema instalado no servidor.

#Servidor sem internet, chamamos de INTRANET

DNS ( Domain Name Service)

  • Cada máquina tem um IP
  • o DNS pega o nome e redireciona para o IP

Hospedagems de sites

Colocar os arquivos do site num servidor (web server) pros clientes poderem acessar.

  • Contrata um servidor de hospedagem
  • Dá para registrar o domínio no mesmo lugar
  • Quando a conexão é segura aparece HTTPs (Criptografada de ponta-aponta)
  • O servidor pode ser de mais de um tipo (arquivos, streaming, proxy, e-mails)

    • Conexão FTP (File transfer protecol)

    • É uma conexão para enviar e receber arquivos.
    • Conexão direta com o servidor.
    #FileZilla: coloca usuário e senha, endereço e conecta com o servidor para mandar os arquivos para o site.
